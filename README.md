Article Sharing Platform

A desktop-based Article Sharing Platform developed using Python (Tkinter) and MySQL that allows users to create, manage, and interact with articles through a user-friendly graphical interface. The application implements user authentication, article management, likes, comments, and CRUD operations while demonstrating core Database Management System (DBMS) concepts and database connectivity.

Features

- User Registration and Login
- Secure Authentication System
- Create, View, Update, and Delete Articles
- Like and Comment Functionality
- User-Specific Access Control
- MySQL Database Integration
- Interactive GUI using Tkinter
- Relational Database Design
- CRUD Operations

Tech Stack

- Python
- Tkinter
- MySQL
- SQL
- MySQL Connector
